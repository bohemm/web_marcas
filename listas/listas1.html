listas

Las listas son elementos que permiten la estructuracion de documentos que pueden organizar total o parcialmente un documento.
El lenguaje HTML permite representar diferentes tipos de listas:
  • listas descriptivas
  • listas regulares
  • listas anidadas
    • Consisten en una serie de elementos que componen la lista, de cada uno de ellos se realiza una definicion o una descripcion.
    Se usa para definir terminos, descripcion de conceptos, etc.

    Listas regulares: Estan formadas por un conjunto de elementos a modo de indice o relacion, y que pueden estar ordenadas o no.
